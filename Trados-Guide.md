# SDL Trados Studio: Quick Start Guide for Linguists

**Compatible with:** SDL Trados Studio 2021/2022/2024

## 📌 Overview

This guide provides a simplified workflow for professional translators to set up their first translation project in **SDL Trados Studio**. It covers the essential steps from file import to project completion.

---

## 🛠 Project Setup Workflow

### 1. Preparing the Environment

Before starting, ensure you have the following:

* **Source File:** Supported formats (e.g., .docx, .idml, .pdf, .html).
* **Translation Memory (TM):** An existing `.sdltm` file or create a new one.
* **Termbase (Optional):** A `.sdltb` file for consistent terminology.

### 2. Creating a New Project

1. Launch **SDL Trados Studio**.
2. Switch to the **Welcome** view.
3. Drag and drop your source file(s) into the main dashboard.
4. Select **Create New Project** (or *Translate as a Single Document* for quick tasks).
5. Set your **Source** and **Target** languages accurately.

### 3. Translation & Segmentation

Once the Editor opens:

* Segments are displayed on the left (Source) and right (Target).

* **Confirming Segments:** Press `Ctrl + Enter` to commit a segment to the TM and move to the next one.

* **Tags:** Ensure all formatting tags (purple icons) are mirrored in the target segment to maintain document layout.

### 4. Finalizing the Project

1. Run the **QA Checker** (`F8`) to identify missing tags or inconsistent terminology.
2. Go to **File** > **Save Target As** to generate the final translated document in its original format.

> [!TIP]
> Use the **Concordance Search** (`F3`) to see how specific terms were translated in previous projects stored in your TM.
